## Git = Gestion de version en local
     Git est un outil qui permet de suivre l’évolution d’un projet sur ton ordinateur.

      Avec Git, tu peux :

        -enregistrer des versions de ton projet (commits)

        -revenir en arrière si tu as fait une erreur

        -comparer les modifications entre deux versions

        -travailler avec des branches pour tester des fonctionnalités

        -fusionner plusieurs versions du code

## GitHub = Hébergement distant + collaboration

       GitHub est une plateforme en ligne qui permet de :

          -stocker ton projet Git sur Internet (dans un repository)

          -partager ton projet avec d’autres développeurs

          -collaborer en équipe

          -gérer des branches à distance

          -faire des Pull Requests (PR) pour proposer des modifications

          -sécuriser ton code avec des sauvegardes
## Différence entre Git et GitHub

Git est un outil de gestion de versions installé sur ton ordinateur qui permet d’enregistrer et suivre les modifications d’un projet, tandis que GitHub est une plateforme en ligne qui héberge les projets Git sur Internet et facilite le partage et la collaboration entre développeurs.
